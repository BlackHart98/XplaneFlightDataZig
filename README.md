# Xplane MFD Calculations in Zig
This repository is largely inspired by this project https://github.com/LaurieWired/XplaneFlightData

## Build
1. To build the calculators run the command below:
```sh
sh ./build.sh
```


## Run
```sh
# Launch the MFD (requires X-Plane with Web API)
sh ./run_mfd.sh
```
